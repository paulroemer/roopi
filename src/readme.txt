RooPie.ROI is the java class that other programs use to talk via RXTX
with the roomba. - it runs on the rpi.

RooPie.GuiServer is an aplication that runs on the rpi and uses
RooPie.ROI.  Run this from the rpi's command line if you want to use
the RoombaGUI.UserInterface to control your roomba from another
machine.
